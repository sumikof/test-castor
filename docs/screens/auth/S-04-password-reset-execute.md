## 要素カタログ

### ページヘッダー

| 要素名 | 型 | data-testid | 必須 | 説明 |
|---|---|---|---|---|
| ページタイトル | text-display | `password-reset-exec-title` | — | 「新しいパスワードの設定」固定テキスト |

### フォーム入力（トークン有効時）

| 要素名 | 型 | data-testid | 必須 | バリデーション | 備考 |
|---|---|---|---|---|---|
| 新しいパスワード | password-input | `password-reset-exec-new-password` | yes | 空文字不可、パスワードポリシー準拠 | type="password"、autocomplete="new-password" |
| パスワード確認 | password-input | `password-reset-exec-confirm-password` | yes | 新しいパスワードと一致 | UI専用（API送信対象外） |

### アクションボタン / リンク

| 要素名 | 型 | data-testid | 条件 | 動作 |
|---|---|---|---|---|
| 設定ボタン | button (primary) | `password-reset-exec-submit` | 両フィールド入力済み・バリデーション通過時に有効 | API 呼び出し → 成功で S-02 にリダイレクト |

### トークン無効時の要素

| 要素名 | 型 | data-testid | 表示条件 | 動作 |
|---|---|---|---|---|
| エラーメッセージ | alert (error) | `password-reset-exec-invalid-token` | トークン検証失敗時 | — |
| パスワードリセットを再要求 | link | `password-reset-exec-retry` | トークン無効時 | S-03 に遷移 |
| ログインに戻る | link | `password-reset-exec-back-to-login` | トークン無効時 | S-02 に遷移 |

## 状態バリエーション

| 状態 | 条件 | 画面の変化 |
|---|---|---|
| トークン検証中 | ページロード中 | ローディング表示 |
| トークン有効 | トークン検証成功 | パスワード入力フォーム表示 |
| トークン無効 | トークン不存在・期限切れ | エラーメッセージ + リンク表示。フォームは非表示 |
| 入力中 | パスワード入力済み | 設定ボタン enabled |
| 送信中 | API リクエスト中 | 設定ボタン loading 状態 |
| 成功 | パスワード設定完了 | S-02 にリダイレクト（トースト付き） |
| エラー（422） | バリデーション失敗 | インラインエラー表示 |

---

## API 呼び出し

### ページロード時

| メソッド | パス | トリガー | レスポンス処理 |
|---|---|---|---|
| GET | `/api/v1/auth/password-reset/:token/validate`（想定） | 画面マウント時 | 有効ならフォーム表示、無効ならエラー画面 |

> API 未定義。実装時にトークン検証エンドポイントの設計が必要。

### フォーム送信時

| メソッド | パス | トリガー | リクエストボディ |
|---|---|---|---|
| POST | `/api/v1/auth/password-reset/:token/execute`（想定） | 設定ボタンクリック | `{ new_password }` |

> API 未定義。URL パスの `:token` はリセットトークン。

**レスポンス処理（想定）:**

| ステータス | 処理 |
|---|---|
| 200 | 成功トースト → S-02 にリダイレクト |
| 400/410 | トークン無効・期限切れ → エラー画面に切替 |
| 422 `VALIDATION_FAILED` | パスワードポリシー不適合 → インラインエラー表示 |

**副作用（想定）:**
- パスワード更新後、対象ユーザーの全セッションを無効化

## 画面遷移

| アクション | 遷移先 | 条件 |
|---|---|---|
| パスワード設定完了 | S-02 ログイン (`/login`) | API 成功時 |
| パスワードリセットを再要求 | S-03 パスワードリセット要求 (`/password-reset`) | トークン無効時のリンククリック |
| ログインに戻る | S-02 ログイン (`/login`) | トークン無効時のリンククリック |

## エラー状態

### フィールドバリデーション（クライアント側）

| フィールド | エラー条件 | エラーメッセージ例 |
|---|---|---|
| 新しいパスワード | 空文字 | 「パスワードを入力してください」 |
| 新しいパスワード | ポリシー不適合 | 「パスワードは8文字以上で入力してください」 |
| パスワード確認 | パスワードと不一致 | 「パスワードが一致しません」 |

### API エラー（想定）

| 状況 | エラーメッセージ例 |
|---|---|
| トークン無効 | 「このリセットリンクは無効です」 |
| トークン期限切れ | 「このリセットリンクは期限切れです。再度リセットを要求してください」 |
| パスワードポリシー不適合 | フィールドごとのインラインエラー |
