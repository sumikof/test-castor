## 要素カタログ

### 表示要素

| 要素名 | 種別 | data-testid | 説明 |
|---|---|---|---|
| ページタイトル | テキスト（h1） | `user-list-title` | 「ユーザー管理」固定 |
| ユーザーテーブル | table | `user-table` | ユーザー一覧テーブル |
| 空状態メッセージ | テキスト | `user-list-empty` | ユーザーが 0 件のとき（通常到達しない — admin が必ず 1 名以上存在） |

### テーブル列

| 列名 | data-testid | 内容 | 備考 |
|---|---|---|---|
| 表示名 | `user-display-name` | `display_name` | — |
| メール | `user-email` | `email` | — |
| ロール | `user-role` | `role` のバッジ表示（admin/editor/viewer） | 色分けバッジ |
| 最終ログイン | `user-last-login` | 最終ログイン日時 | **※ API 未定義** — `GET /api/v1/users` のレスポンスに `last_login` がない。Session テーブルの最新 `created_at` から派生するか、User テーブルに列追加が必要 |

### テーブル行

|---|---|---|---|
| ユーザー行 | table row (clickable) | `user-row-{id}` | クリックで S-19 編集ダイアログを開く |

### 操作要素

## 状態バリエーション

### ロールバッジの表示

| ロール | バッジ色 | ラベル |
|---|---|---|
| admin | 赤系 | 「管理者」 |
| editor | 青系 | 「編集者」 |
| viewer | グレー系 | 「閲覧者」 |

### 自分自身の行

- 現在ログイン中のユーザーの行には視覚的なインジケータ（例: 行ハイライト、「あなた」ラベル）を表示
- data-testid: `user-row-current`

## API 呼び出し

| トリガー | メソッド | パス | リクエスト | レスポンス処理 |
|---|---|---|---|---|
| ページ読み込み | GET | `/api/v1/users` | — | テーブルにユーザー一覧を表示 |

### GET レスポンス

```json
{
  "items": [
    {
      "id": "uuid",
      "email": "tanaka@example.com",
      "display_name": "田中太郎",
      "role": "admin",
      "created_at": 1719388800000,
      "updated_at": 1719388800000
    }
  ]
}
```

> **API ギャップ:** ワイヤーフレームの「最終ログイン」列に対応するフィールドが `GET /api/v1/users` レスポンスにない。実装時に `last_login_at` フィールドの追加、または Session テーブルからの派生を検討する必要がある。

## 画面遷移

| アクション | 遷移先 | 備考 |

## エラー状態

| エラー | HTTP | 表示方法 |
|---|---|---|
| 認証切れ | 401 | ログイン画面（S-02）にリダイレクト |
| 権限不足（admin 以外） | 403 | 画面自体にアクセスできない。グローバルヘッダーに「ユーザー管理」リンクを表示しない |
| ユーザー一覧取得失敗 | 5xx | エラーメッセージ表示 + リトライボタン |
