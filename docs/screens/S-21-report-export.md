## 要素カタログ

### 表示要素

| 要素名 | 種別 | data-testid | 説明 |
|---|---|---|---|
| ページタイトル | テキスト（h1） | `report-title` | 「レポート」固定 |
| ステータス別集計セクション | セクション | `report-status-summary` | 横棒グラフ + 数値 |
| approved 件数 | テキスト | `report-status-approved` | approved の件数 |
| draft 件数 | テキスト | `report-status-draft` | draft の件数 |
| archived 件数 | テキスト | `report-status-archived` | archived の件数 |
| ステータス別グラフ | 棒グラフ | `report-status-chart` | ステータス別の横棒グラフ |
| カテゴリ別分布セクション | セクション | `report-category-summary` | カテゴリ別の件数表示 |
| normal 件数 | テキスト | `report-category-normal` | 正常系の件数 |
| abnormal 件数 | テキスト | `report-category-abnormal` | 異常系の件数 |
| boundary 件数 | テキスト | `report-category-boundary` | 境界値の件数 |
| error_handling 件数 | テキスト | `report-category-error-handling` | エラーハンドリングの件数 |
| drift 発生件数 | テキスト | `report-drift-count` | drift 発生中のテストケース数 `[※]` |
| stale 発生件数 | テキスト | `report-stale-count` | stale のテストケース数 `[※]` |

### 操作要素

| 要素名 | 種別 | data-testid | 必須 | バリデーション | 表示条件 |
|---|---|---|---|---|---|
| プロジェクトフィルタ | select (dropdown) | `report-project-filter` | — | — | 常時。選択肢:「全て」+ 各プロジェクト名 |
| CSV エクスポートボタン | button | `report-csv-export` | — | — | 常時 |

## 状態バリエーション

### プロジェクトフィルタの効果

| 選択値 | 表示内容 |
|---|---|
| 全て | 全プロジェクトの集計値 |
| 特定プロジェクト | 選択したプロジェクトのみの集計値 |

### データなし状態

- テストケースが 0 件の場合: グラフと数値を 0 で表示
- プロジェクトが 0 件の場合:「プロジェクトを作成してください」メッセージ

## API 呼び出し

> **※ API 未定義:** レポート・集計用の専用 API エンドポイントは未実装。以下は実装時に必要になる想定 API。

| トリガー | メソッド | パス（案） | リクエスト | レスポンス処理 |
|---|---|---|---|---|
| ページ読み込み | GET | `/api/v1/reports/summary` | `?project_id=` | ステータス別・カテゴリ別集計を表示 |
| プロジェクトフィルタ変更 | GET | `/api/v1/reports/summary` | `?project_id=:pid` | 集計値を再取得 |
| CSV エクスポートボタン | GET | `/api/v1/reports/export` | `?project_id=&format=csv` | CSV ファイルのダウンロード |

### 想定レスポンス構造（案）

```json
{
  "status_summary": {
    "approved": 280,
    "draft": 50,
    "archived": 12
  },
  "category_summary": {
    "normal": 120,
    "abnormal": 95,
    "boundary": 80,
    "error_handling": 47
  },
  "drift_count": 8,
  "stale_count": 4,
  "total": 342
}
```

## エラー状態

| エラー | HTTP | 表示方法 |
|---|---|---|
| 認証切れ | 401 | ログイン画面（S-02）にリダイレクト |
| 集計 API 失敗 | 5xx | エラーメッセージ表示 + リトライボタン |
| CSV エクスポート失敗 | 5xx | エラートースト |

## トースト・通知メッセージ

| 条件 | 種別 | メッセージ例 |
|---|---|---|
| CSV エクスポート成功 | success | 「CSV ファイルをダウンロードしました」 |
| CSV エクスポート失敗 | error | 「エクスポートに失敗しました」 |
