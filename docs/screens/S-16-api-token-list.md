## 要素カタログ

### 表示要素

| 要素名 | 種別 | data-testid | 説明 |
|---|---|---|---|
| ページタイトル | テキスト（h1） | `token-list-title` | 「API トークン」固定 |
| トークンテーブル | table | `token-table` | トークン一覧テーブル |
| 注釈テキスト | テキスト | `token-list-note` | 「トークンの平文は発行時に1回だけ表示されます」 |
| 空状態メッセージ | テキスト | `token-list-empty` | トークンが 0 件のとき表示。「APIトークンがありません」 |

### テーブル列

| 列名 | data-testid | 内容 | ソート |
|---|---|---|---|
| 名前 | `token-name` | トークンのラベル | — |
| 作成日 | `token-created-at` | `created_at` を日付フォーマット | — |
| 最終使用 | `token-last-used` | `last_used_at` を日付フォーマット。未使用時は「未使用」 | — |
| 状態 | `token-status` | `revoked_at` の有無で判定。有効 / 失効済み バッジ表示 | — |
| 操作 | `token-actions` | 失効ボタン（有効なトークンのみ） | — |

### 操作要素

| 要素名 | 種別 | data-testid | 必須 | バリデーション | 表示条件 |
|---|---|---|---|---|---|
| 発行ボタン | button | `token-issue-button` | — | — | 常時表示 |
| 失効ボタン（行ごと） | button | `token-revoke-button-{id}` | — | — | `revoked_at` が null（有効）のトークンのみ |

### 失効確認ダイアログ（インライン）

| 要素名 | 種別 | data-testid | 説明 |
|---|---|---|---|
| 確認メッセージ | テキスト | `token-revoke-confirm-message` | 「トークン「{name}」を失効させますか？失効後は即座に認証が拒否されます」 |
| キャンセルボタン | button | `token-revoke-confirm-cancel` | ダイアログを閉じる |
| 失効実行ボタン | button | `token-revoke-confirm-execute` | 失効 API を呼ぶ |

---

## 状態バリエーション

### トークン状態の表示

| 条件 | 状態バッジ | 失効ボタン |
|---|---|---|
| `revoked_at === null` | 「有効」（緑バッジ） | 表示 |
| `revoked_at !== null` | 「失効済み」（グレーバッジ） | 非表示 |

### トークン 0 件

- テーブル非表示
- 空状態メッセージ表示:「APIトークンがありません。「発行」ボタンからトークンを作成してください」

## API 呼び出し

| トリガー | メソッド | パス | リクエスト | レスポンス処理 |
|---|---|---|---|---|
| ページ読み込み | GET | `/api/v1/projects/:pid/tokens` | — | テーブルにトークン一覧を表示 |
| 失効確認で「失効」クリック | DELETE | `/api/v1/projects/:pid/tokens/:id` | — | テーブル更新 + トースト |

### GET レスポンス

```json
{
  "items": [
    {
      "id": "uuid",
      "name": "discovery-ci",
      "created_at": 1719388800000,
      "revoked_at": null,
      "last_used_at": 1719450000000
    }
  ]
}
```

### DELETE レスポンス

```json
{
  "id": "uuid",
  "name": "discovery-ci",
  "revoked_at": 1719460000000
}
```

## エラー状態

| エラー | HTTP | 表示方法 |
|---|---|---|
| 認証切れ | 401 | ログイン画面（S-02）にリダイレクト |
| 権限不足（admin 以外） | 403 | 画面自体にアクセスできない。ナビに表示しない |
| プロジェクト不存在 | 404 | 404 ページ表示 |
| 失効 API 失敗 | 5xx | エラートースト |
