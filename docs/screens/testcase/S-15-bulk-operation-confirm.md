## 要素カタログ

### 表示要素

| 要素名 | 種別 | data-testid | 説明 |
|---|---|---|---|
| ダイアログタイトル | テキスト | `bulk-confirm-title` | 「一括操作の確認」固定 |
| 対象件数テキスト | テキスト | `bulk-confirm-count` | 「{N} 件のテストケースを」 |
| アクション名テキスト | テキスト | `bulk-confirm-action-name` | 「承認」/「アーカイブ」/「復帰」 |
| 警告テキスト（approve 時） | テキスト | `bulk-confirm-warning-ownership` | machine→human 遷移の注意 |
| 警告テキスト（approve 時） | テキスト | `bulk-confirm-warning-auto-update` | 自動更新停止の注意 |
| 警告テキスト（archive 時） | テキスト | `bulk-confirm-warning-archive` | アーカイブ後もデータは保持される旨 |
| 警告テキスト（restore 時） | テキスト | `bulk-confirm-warning-restore` | draft に復帰し再レビューが必要な旨 |

### 操作要素

| 要素名 | 種別 | data-testid | 必須 | バリデーション | 表示条件 |
|---|---|---|---|---|---|
| キャンセルボタン | button | `bulk-confirm-cancel` | — | — | 常時表示 |
| 実行ボタン | button | `bulk-confirm-execute` | — | — | 常時表示。リクエスト中は disabled + ローディング |

---

## 状態バリエーション

### アクション別の表示内容

| action | ダイアログ文言 | 警告テキスト |
|---|---|---|
| `approve` | 「{N} 件のテストケースを「承認」しますか？」 | machine→human 遷移の注意、自動更新停止の注意 |
| `archive` | 「{N} 件のテストケースを「アーカイブ」しますか？」 | 物理削除ではない旨。archived フィルタで復帰可能 |
| `restore` | 「{N} 件のテストケースを「復帰」しますか？」 | draft に戻り再レビューが必要な旨 |

### ボタン状態

| 状態 | 実行ボタン | キャンセルボタン |
|---|---|---|
| 初期表示 | enabled | enabled |
| リクエスト中 | disabled + スピナー | disabled |
| 完了 | ダイアログ閉じる | — |

---

## API 呼び出し

| トリガー | メソッド | パス | リクエスト | レスポンス処理 |
|---|---|---|---|---|
| 実行ボタンクリック | POST | `/api/v1/projects/:pid/testcases/bulk` | `{ ids: string[], action: "approve"\|"archive"\|"restore" }` | 成功: ダイアログ閉じる + S-08 リスト再取得 + トースト |

### リクエスト詳細

```json
{
  "ids": ["uuid-1", "uuid-2", "uuid-3"],
  "action": "approve"
}
```

- `ids`: S-08 で選択されたテストケースの ID 配列（最大 100 件）
- `action`: `"approve"` / `"archive"` / `"restore"`

### レスポンス詳細

```json
{
  "updated": 15,
  "skipped": 2,
  "errors": [
    { "id": "uuid-16", "code": "OCC_CONFLICT", "message": "version mismatch" }
  ]
}
```

## 画面遷移

| アクション | 遷移先 | 備考 |
|---|---|---|
| キャンセルボタン | S-08（ダイアログ閉じる） | 選択状態は維持 |
| 実行成功 | S-08（ダイアログ閉じる + リスト再取得） | 選択状態はクリア |
| オーバーレイ背景クリック | S-08（ダイアログ閉じる） | キャンセルと同じ |
| Escape キー | S-08（ダイアログ閉じる） | キャンセルと同じ |

---

## エラー状態

| エラー | 表示方法 |
|---|---|
| 全件成功 | トーストのみ（エラー表示なし） |
| 一部失敗（OCC 競合等） | トーストに updated/skipped/error 件数を表示 |
| 全件失敗 | エラートーストで件数表示 |
| ネットワークエラー | エラートーストで再試行を促す |

---

## トースト・通知メッセージ

| 条件 | 種別 | メッセージ例 |
|---|---|---|
| 全件成功 | success | 「{N} 件のテストケースを承認しました」 |
| 一部スキップ | success | 「{updated} 件を承認しました（{skipped} 件はスキップ）」 |
| 一部エラー | warning | 「{updated} 件を承認しました（{errors.length} 件でエラー発生）」 |
| ネットワークエラー | error | 「一括操作に失敗しました。再度お試しください」 |
